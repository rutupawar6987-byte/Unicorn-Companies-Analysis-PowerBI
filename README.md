# 🦄 Unicorn Companies Analysis Dashboard

## 📊 Project Overview
This project analyzes unicorn companies using Power BI to uncover insights about valuation, funding, industries, and global distribution.

## 🚀 Features
- KPI cards for Total Companies, Valuation, and Funding
- Industry-wise analysis
- Country-wise distribution (map)
- Year-wise trends
- Interactive filters

## 🛠️ Tools Used
- Power BI
- Excel
- DAX
- Data Cleaning

## 📌 Key Insights
- North America has the highest number of unicorn companies
- FinTech and Internet Software dominate
- Rapid growth after 2020
